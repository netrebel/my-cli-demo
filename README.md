# my-cli-demo

https://github.com/spf13/cobra-cli/blob/main/README.md

# How to run

* `go run main.go serve`
* `go run main.go config`
* `go run main.go config create`

# How to build executable

```
go build
```

and then:

* `my-cli-demo serve`
* `my-cli-demo config`
* `my-cli-demo config create`
